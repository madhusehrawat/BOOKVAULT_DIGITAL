# 📚 Book Vault | Premium Digital Library

A professional full-stack E-commerce platform and digital asset library built with the **Node.js MVC architecture**. This application features secure user authentication, a premium subscription model, and an administrative dashboard for real-time metrics.

## 🚀 Key Features

* **Premium Membership System:** Tiered access to digital assets. Premium books are locked behind a subscription wall with custom UI alerts.
* **Secure Authentication:** Implementation of JWT-based sessions, password hashing, and "Forgot Password" flows via GMAILAPI.
* **Admin Dashboard:** Real-time monitoring of BOOKS, MEMBER , and COMMUNITY .
* **Responsive UI:** Crafted with **Tailwind CSS** and **EJS** for a modern, professional aesthetic.
* **Library Management:** Users can "Like" books to add them to their personal collection and share assets via the Web Share API.
* **Optimized Downloads:** Secure, cookie-aware PDF download system to prevent unauthorized access.
* **community system :** create community , join community and group disussion 
## 🛠️ Tech Stack

**Frontend:** EJS (Embedded JavaScript), Tailwind CSS, SweetAlert2, Toastify  
**Backend:** Node.js, Express.js  
**Database:** MongoDB Atlas (Mongoose ODM)  
**Deployment:** Render.com
